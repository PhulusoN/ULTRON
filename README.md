# ULTRON

ULTRON - Music Management App
ULTRON is an Android application built with Jetpack Compose for managing music playlists and viewing track details.
Features
•
Main Dashboard: Entry point with navigation to add songs or view details.
•
Add to Playlist: A form to capture song titles, artist names, ratings, and comments.
•
Detailed View: A dedicated screen for viewing specific song information.
•
Material 3 UI: Uses modern components like ElevatedButtons and OutlinedTextFields.
Technical Stack
•
Language: Kotlin
•
Framework: Jetpack Compose
•
Design System: Material Design 3
•
Navigation: Activity-based navigation using Intents
